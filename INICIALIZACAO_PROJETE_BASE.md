# Comandos Basicos

## Inicializar um projeto node - Cria arquivo package.json.
~~~~javascript
npm init -y
~~~~

## Instalação do TYPESCRIPT e TYPES
~~~~javascript
npm i typescript @types/node --save-dev
~~~~

### Incialização do Typesript
~~~~javascript
npx tsc --init
~~~~

Importar e Exportar 

~~~~javascript
export * from './functions/valida-matricula';
~~~~
